docomo LIVE UX バックアップ 使い方ガイド

アプリ概要

「docomo LIVE UX バックアップ」はホームアプリ「docomo LIVE UX」においてお客様が設定されているアプリケーションのアイコンの配置、ウィジェットの配置およびホーム画面数等の情報を当社のドコモクラウドを用いてバックアップ、復元およびマルチデバイス同期を行うアプリです。


<主な機能>

  • 手動バックアップ
  • 自動バックアップ
  • バックアップデータの復元
  • マルチデバイス同期

使い方


    ① バックアップ（手動バックアップ）

    メニュー画面右上の「バックアップ」ボタンを押下することにより即時にバックアップが開始されます。


    ② 自動バックアップ

    「自動バックアップ」の右にあるチェックボックスにチェックを入れることで設定が有効となります。（再びタップし、チェックを外すことで設定は無効となります。）

    本設定を有効に設定すると、1日1回バックアップが自動で行われます。

    （但し、最新のバックアップデータとアプリアイコン等の配置情報に差分がない場合は、自動バックアップは行われません。）

    [補足] 　圏外など、ご利用される通信環境によってはバックアップデータが保存されないことがあります。


    ③ マルチデバイス同期

    「マルチデバイス同期」の右にあるチェックボックスにチェックを入れることで設定が有効となります。（再びタップし、チェックを外すことで設定は無効となります。）

    本機能を有効に設定すると、本端末に新たにインストールしたアプリのアイコンが、同一の"dアカウントのID"が設定された別の端末にも 黒みがかったアイコンとして追加されます。アプリ本体は複製されませんが作成されたアイコンをタップいただくことでアプリのダウンロードが可能です。

    [補足] 　アイコンが複製される側の端末につきましても本設定を有効設定にしておく必要があります。


    ④ Wi-Fi通信設定

    「Wi-Fi通信設定」の右にあるチェックボックスにチェックを入れることで設定が有効となります。（再びタップし、チェックを外すことで設定は無効となります。）

    本設定を有効に設定すると、Wi-Fi接続時のみ"バックアップ"や"マルチデバイス同期"を行うことが可能となります。

    [補足] 　Wi-Fi接続が行われていない状態ではネットワークエラーとなりバックアップデータの保存を行うことができません。


    ⑤ 海外ローミング設定

    「海外ローミング設定」の右にあるチェックボックスにチェックを入れることで設定が有効となります。（再びタップし、チェックを外すことで設定は無効となります。）

    日本国外でローミング接続をされる場合に本設定を有効設定することにより、"バックアップ"や"マルチデバイス同期"を行うことが可能となります。

    [補足] 　海外ローミング時に利用するには、本設定項目に加え、端末本体設定のデータローミング設定が有効となっている必要があります。


    ⑥ 利用情報の送信

    「利用情報の送信」の右にあるチェックボックスにチェックを入れることで有効となります。

    本機能を有効にした場合、利用者情報の送信が行われます。送信される"情報"及び"利用目的"については『利用規約』をご確認ください。

    (利用情報の提供をご希望されない場合は、チェックボックスのチェックを外して頂くことにより機能を無効にできます。)


    ⑦ バックアップデータの削除

    ドコモクラウド上に保存されたバックアップデータの削除が行えます。

    "バックアップデータの削除"を押下するとドコモクラウド上に保存されたバックアップデータ一覧が表示されますので、削除したいデータにチェックを入れ、画面下部の削除ボタンを押下することでバックアップデータの削除を行うことができます。

    [ご注意事項]

    • 削除されたバックアップデータは元に戻すことができません。
    • データ削除後も自動バックアップ設定がONの場合、配置情報が更新された際には再びバックアップが実施されます。
      • （配置情報に変更がない場合は自動バックアップが実施されませんので、一度手動バックアップを実施してください。）



バックアップデータの復元について

「docomo LIVE UX バックアップ」を起動された際、ドコモクラウド上にバックアップデータが保存されていますと「最新のバックアップデータを復元」ボタンが表示されますので押下することでバックアップデータの復元を行うことができます。

また、複数のバックアップデータが保存されていた場合は「バックアップデータ一覧から復元」ボタンが表示され、押下すると保存されているバックアップデータがリスト表示されますので、こちらから復元を行うバックアップデータを選択し復元いただくこともできます。

[ご注意事項]

  • バックアップを行った端末への同一データの復元は行うことができません。

    • （端末初期化を行った場合は復元可能となります。）

  • 対応機種でない場合など、一部、復元できないアプリやウィジェットがあります。
  • ウィジェットによっては、復元した際にアイコン表示に切り替わる場合があります。アイコンをタップすることでウィジェットの復元が可能となります。
  • 「dアカウント設定」アプリに必要なアクセス権限が未許可の場合には、docomo LIVE UX バックアップをご利用いただけない可能性があります。
